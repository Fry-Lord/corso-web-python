📘 Tutorial Flask – Guida Completa

1. Cos’è Flask

Flask è un framework scritto in Python che permette di creare applicazioni web dinamiche.

In pratica, Flask permette a Python di comunicare con il browser.

  • Riceve richieste dal browser
  • Elabora dati (con Python)
  • Restituisce una risposta (HTML)
Flask è il collegamento tra il mondo del web (HTML) e la logica (Python)

2. Server e Localhost

Un server è un programma che risponde alle richieste del browser.

Quando scrivi un indirizzo web, il browser invia una richiesta a un server.

Browser → richiesta → Server → risposta → Browser

Con Flask, il server gira sul tuo computer:

http://127.0.0.1:5000
  • 127.0.0.1 = il tuo computer (localhost)
  • 5000 = porta di comunicazione
Localhost significa: il sito è eseguito sul tuo PC, non su Internet

12. GET e POST + esempio

I metodi HTTP definiscono come il browser comunica con il server.

  • GET: richiede dati (es. aprire una pagina)
  • POST: invia dati (es. form)
GET = leggere dati
POST = inviare dati

app.py

from flask import Flask, render_template, request

app = Flask(__name__)

@app.route("/", methods=["GET","POST"])
def home():

    risultato = None

    if request.method == "POST":
        a = int(request.form["a"])
        b = int(request.form["b"])
        risultato = a + b

    return render_template("index.html", risultato=risultato)

app.run(debug=True)
